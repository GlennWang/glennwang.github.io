Python range() 函數用法

簡介

range() 函數是 Python 中的一個函數，用於生成一個整數序列，通常在迴圈中使用。該函數返回的是一個特殊的物件，稱為可迭代物件（iterable object）。

要理解什麼是可迭代物件，最快速的就是放出一個可迭代物件給你看，比如：

my_list = [1, 2, 3, 4, 5]

list和string就是可迭代的，可以用以下方式迭代：

for item in my_list:
    print(item)

range() 函數的可迭代物件是一種可以逐個訪問元素的物件，但它並不提前在內存中存儲所有元素，而是在需要時按需生成。這樣可以節省空間，尤其是在處理大型範圍時。

語法

range(stop)
range(start, stop[, step])
  • range()預設的start是0，所以range(10)等於range(0,10)。
  • range()的計數到stop，但不包含stop，range(0,4)是0到3，沒有4。
  • range()的step就是一次走多少步，預設是1，所以如果要0 2 4的話要寫range(0,5,2)

基礎用法

range(stop=5)

for i in range(5):
    print(i)

上面的程式碼將輸出：

0
1
2
3
4

range(start=1, stop=5)

for i in range(1, 5):
    print(i)

上面的程式碼將輸出：

1
2
3
4

range(start=0, stop=5[, step=2])

for i in range(0, 5, 2):
    print(i)

上面的程式碼將輸出：

0
2
4

range(start=5, stop=0[, step=-1])

也可以反過來輸出，不過start的值要大於stop的值。

for i in range(5, 0, -1):
    print(i)

上面的程式碼將輸出：

5
4
3
2
1

細節

總之，range() 返回的是一個用於表示整數序列的可迭代物件。在python2.x中，他返回的是一個list，但在python3中不是。

如果你需要將其轉換為list，可以使用 list() 函數：

my_list = list(range(5))
print(my_list)

這將輸出：

[0, 1, 2, 3, 4]
